# Site da Conplena

Site institucional estático, pronto para publicação gratuita no GitHub Pages.

## Publicação no GitHub

1. Abra o repositório `conplena-site`.
2. Clique em **Add file > Upload files**.
3. Arraste todo o conteúdo desta pasta, mantendo as pastas `assets`.
4. Confirme o commit.
5. Vá em **Settings > Pages**.
6. Em **Build and deployment**, selecione **Deploy from a branch**.
7. Selecione a branch `main` e a pasta `/ (root)`.
8. Salve.

## Antes de publicar

Atualize em todos os arquivos, quando disponível:
- Instagram definitivo
- E-mail profissional
- Endereço completo
- CRC
- Foto atual na página Nossa História

## Domínio próprio

Depois que o GitHub Pages estiver funcionando, conecte `conplena.com.br` em **Settings > Pages > Custom domain**.
